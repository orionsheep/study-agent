# Demo Script

1. Start backend and web with `bash scripts/run_full_validation.sh` or run the services manually.
2. Show the left Spatial Learning App Canvas and right Tutor Chat.
3. Ask: `我是软件工程大一，Python 一般，数学推导弱，喜欢图解和代码。`
4. Ask for `神经网络学习路径` and click the AppLink Flight chip.
5. Open WorkEnergyDemoApp, change sliders, and ask the Tutor to explain current values.
6. Submit a wrong quiz answer and show misconception/mastery evidence on the Dashboard.
7. Click `总结到笔记 App` and show the editable Notes App.
8. Open system status and explain external readiness blockers.
