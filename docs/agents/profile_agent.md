# Profile Agent

File: `services/api/app/agents/profile_agent.py`

Extracts major, grade, foundation, goal, cognitive style, pace, weak points, interests, preferred resources, and mastery map signals from natural language, then writes EduMem0 profile memories.

Tests: `test_profile_agent_extracts_many_dimensions`, `test_chat_profile_memory_and_dashboard_evidence`.
