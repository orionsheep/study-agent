# App Canvas Design

The main canvas is a custom spatial App window manager, not React Flow. It supports pan, zoom, drag, resize, frames, relationship connectors, App Dock, search, minimap, auto arrange, save/restore, focus, split, fullscreen, minimized, and AppLink Flight.

Right-side chat remains process-oriented; persistent learning artifacts live as Apps on the left canvas. AppLink clicks animate from chat to target App, focus the App, persist the link event, and write app interaction memory.

React Flow is reserved only for graph-style Apps if added later; the current graph renderers use lightweight SVG.
